## ğŸ› Common Issues & Solutions

### App Won't Start
```
Problem: Double-clicking exe does nothing
Solution:
1. Check if app is already running (Task Manager)
2. Try right-click â†’ Run as Administrator
3. Delete %APPDATA%\dvault\cache\ folder
4. Try again
```

### "Window Cannot Be Found"
```
Problem: App says interface won't load
Solution:
1. Close app completely
2. Delete cache folder:
   C:\Users\MikeT\AppData\Roaming\dvault\cache\
3. Restart app
4. Should work now
```

### USB Key Not Recognized
```
Problem: App says "Device not found"
Solution:
1. Unplug USB key
2. Wait 5 seconds
3. Plug back in
4. Restart DVAULT
5. Check Device Manager (Windows + Pause/Break â†’ Device Manager)
   Look under "Universal Serial Bus"
```

### "Insufficient Balance"
```
Problem: Says not enough to send
Reason:
- Balance less than amount + fee
- Or balance shown is cached/old

Solution:
1. Click Refresh button
2. Wait 5 seconds for new balance
3. Try sending lower amount
4. Or add more funds to wallet
```

### Transaction Stuck on "Pending"
```
Problem: Transaction doesn't confirm
Reason: Low network fee
Solution:
1. Wait 24-48 hours (networks are slow sometimes)
2. Or create new transaction with higher fee
3. Check blockchain explorer with transaction ID
4. Contact support if still not confirmed after 48 hours
```

### App Crashes When Sending
```
Problem: App closes during transaction
Solution:
1. Restart app
2. Check if transaction actually sent (in history)
3. If sent, ignore the crash
4. If not sent, retry
5. Try with lower amount first
```
