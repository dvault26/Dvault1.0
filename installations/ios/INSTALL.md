## üêõ TROUBLESHOOTING

### App Won't Install
```
Problem: "This app is not compatible"
Solution:
1. Check iOS version: Settings ‚Üí General ‚Üí About
2. Ensure iOS 13.0 or newer
3. Free up space: Settings ‚Üí General ‚Üí iPhone Storage
4. Try again on App Store
```

### App Crashes on Launch
```
Problem: App opens then closes
Solution:
1. Force quit: Swipe up from bottom (or use control center)
2. Hard restart: Settings ‚Üí General ‚Üí Shutdown ‚Üí Restart
3. Reinstall: Delete and reinstall from App Store
```

### Can't Enable Biometric
```
Problem: Face ID / Touch ID option disabled
Solution:
1. Check iOS has Face ID/Touch ID: Settings ‚Üí Face/Touch ID
2. Enroll your biometric: Settings ‚Üí Face/Touch ID & Passcode
3. Try enabling again in DVAULT settings
```

### Blockchain Connection Failed
```
Problem: "Unable to connect to Bitcoin network"
Solution:
1. Check internet connection: Wi-Fi or cellular
2. Open Settings ‚Üí Wi-Fi (if using Wi-Fi)
3. Close and reopen app
4. Wait 30 seconds for network detection
5. Retry connection in app settings
```

### Transaction Stuck
```
Problem: Send button doesn't work
Solution:
1. Check internet connection
2. Ensure you have sufficient balance
3. Wait 5 minutes
4. Force close and reopen app
5. Try again
```

### Backup Not Working
```
Problem: iCloud backup fails
Solution:
1. Check iCloud Storage: Settings ‚Üí [Your Name] ‚Üí iCloud
2. Free up iCloud space if needed (5GB+ recommended)
3. Ensure Wi-Fi connected
4. Go to Settings ‚Üí [Your Name] ‚Üí iCloud ‚Üí DVAULT
5. Toggle off then on
```

### Performance Issues
```
Problem: App is slow or freezes
Solution:
1. Close background apps: Swipe up in app switcher
2. Restart device: Power off and on
3. Clear app cache: Delete and reinstall (last resort)
4. Update iOS: Settings ‚Üí General ‚Üí Software Update
```
